# git-message-analyst
> Cli templates for javascript.

## template
```sh
# clone the template
git clone git@github.com:aric-tpls/git-message-analyst.git
mvg

# or use tiged
tiged git@github.com:aric-tpls/git-message-analyst.git --mode=git --disable-cache

# replace the template
rifc
```
